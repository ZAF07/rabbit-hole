# Seed Topic Taxonomy

The content classification taxonomy (UI: "Spools"). Deliberately **partial and taste-driven** — a point of view, not an encyclopedia. Topics form a shallow **multi-parent DAG**; a Piece belongs to many Topics; Connections cross Topics freely (see [ADR 0002](adr/0002-taxonomy-and-content-graph-are-separate.md)). Two levels for V1 (category → subcategory); add depth only if browsing demands it.

## V1 cluster — "systems of the modern world"

Opened first because these five **interlock densely**, which maximizes the high-value cross-Topic Connections (the shipping-container story alone touches all five).

## Multi-parent examples (the DAG earning its keep)

- **Behavioral Economics** → *Economics & Markets* + *Psychology & the Mind*
- **Cyber Conflict** → *Technology & Computing* + *Warfare & Strategy*

A subcategory can ship in V1 via one parent even when its other parent category is a deferred wave.
